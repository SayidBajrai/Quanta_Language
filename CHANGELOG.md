# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-15

### Added
- Initial release of Quanta Language
- Lexer and parser for Quanta syntax
- AST representation
- Semantic analysis and validation
- OpenQASM 3 code generation
- CLI tool (`quanta` command)
- Public API (`compile`, `run`)
- Basic gate set: H, X, CNot, CZ, Swap, RZ, Measure
- Function inlining
- For loop unrolling
- Qiskit runtime integration

### Documentation
- README with examples
- Language specification
- Compiler pipeline documentation
